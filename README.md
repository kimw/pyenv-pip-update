# pyenv-pip-update

Update all libraries managed by pip in all versions ([pyenv](https://github.com/pyenv/pyenv) plugin).

## Installation
    git clone https://github.com/massongit/pyenv-pip-update $(pyenv root)/plugins/pyenv-pip-update

## Usage
    pyenv pip-update
